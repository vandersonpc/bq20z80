# Macbook Battery Model A1185 data read

Read the MacBook White (A1181) Internal Battery Texas Instrumens **DQ20Z80** IC using an Arduino Leonardo.

## Battery Connector PinOut

![](A1185_pinout.png)

 Pin       | Description          
| ------------- |:-------------:
| POS     | Positive +
| TERM    | Internal Thermistor     
| SCL     | I2C Clock      
| SDA     | I2C Data  
| SW      | Internal Switch 
| GND     | Ground
